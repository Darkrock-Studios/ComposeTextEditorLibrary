The colors and text style applied to a TextEditor. Any Color.Unspecified field falls back to a sensible default at draw time, so you only set what you want to change.

Prefer rememberTextEditorStyle, which derives these from the active androidx.compose.material3.MaterialTheme color scheme and tracks theme changes. Construct directly only when you need colors independent of the theme.

val blockquoteBackgroundColor: Color

Tinted background fill drawn across the full width of every blockquote line, behind the bar and the text. Color.Unspecified skips the fill entirely — set to a low-alpha color to mark the quoted area as a soft card. The fill is drawn on top of the text (rich spans render after drawText), so use a low alpha to avoid muddying the body.

val codeFenceBackgroundColor: Color

Tinted background fill drawn behind every line in a fenced code block. Color.Unspecified falls back to Color.Gray.copy(alpha = 0.18f). Use a stronger alpha than blockquoteBackgroundColor so the two cards don't read as the same treatment.

val codeFenceBorderColor: Color

Hairline border drawn around the run of fenced code lines — top edge on the first line, bottom edge on the last, sides on every line. Color.Unspecified falls back to Color.Gray.copy(alpha = 0.55f).
