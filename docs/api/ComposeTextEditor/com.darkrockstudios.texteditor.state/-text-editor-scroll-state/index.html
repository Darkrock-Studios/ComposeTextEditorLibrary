TextEditorScrollState

class TextEditorScrollState(initial: Int = 0) : ScrollableState(source)

Vertical scroll position for a text editor, as a Compose ScrollableState.

Position is tracked in pixels and clamped to minValue..maxValue; setting those bounds re-clamps the current value. Use with Modifier.scrollable, or drive it directly via scrollTo, scrollBy, and animateScrollTo.
