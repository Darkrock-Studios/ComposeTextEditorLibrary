TextEditorScrollbar

@Composable
actual fun TextEditorScrollbar(modifier: Modifier, scrollState: TextEditorScrollState, content: (modifier: Modifier) -> Unit)(source)

Wraps editor content with a platform-appropriate vertical scrollbar driven by scrollState. The platform implementation draws a scrollbar where the host conventionally expects one (e.g. desktop) and passes a Modifier to content so it can reserve room for it.

Parameters

modifier

Applied to the scrollbar container.

scrollState

Scroll position the scrollbar reflects and controls.

content

The editor, receiving a Modifier to apply to its root.

iOS implementation of TextEditorScrollbar. Provides a simple visual scrollbar indicator similar to native iOS scroll views. The scrollbar is display-only (non-interactive) following iOS conventions where users scroll by touching the content area, not the scrollbar.
